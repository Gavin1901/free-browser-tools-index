# Free Browser Tools Built With AI

A small index of free, privacy-first browser tools I maintain.

## Time Zone and Meeting Tools

- [EST to IST converter](https://zoneplan.net/abbreviation/est-to-ist/) — quick reference for remote calls between Eastern Time and India Standard Time.
- [Meeting planner](https://zoneplan.net/meeting-planner/) — compare working hours and plan fair meeting times.

## Fitness Calculators

- [Calorie deficit calculator](https://freetdee.com/calorie-deficit/) — estimate a practical calorie target for weight loss planning.
- [TDEE calculator](https://freetdee.com/) — estimate total daily energy expenditure.
- [BMR calculator](https://freetdee.com/bmr-calculator/) — estimate basal metabolic rate.
- [Macro calculator](https://freetdee.com/macro-calculator/) — split daily calories into protein, carbs, and fat.

## HEIC and Live Photo Tools

- [HEIC to JPG converter](https://livephotokit.com/heic-to-jpg/) — convert iPhone HEIC photos to JPG in the browser.
- [Open LIVP file](https://livephotokit.com/open-livp-file/) — open Apple Live Photo packages.
- [Image compressor](https://livephotokit.com/image-compressor/) — compress images without uploading files.
- [LivePhotoKit](https://livephotokit.com/) — browser-side HEIC and Live Photo tools.

## Puppy Vaccine Schedule Tools

- [Puppy vaccine schedule](https://pupvax.com/puppy-vaccine-schedule/) — simple planning reference for first-time dog owners.
- [Labrador vaccine schedule](https://pupvax.com/vaccine-schedule/labrador/) — breed-specific schedule example.
- [Titer testing guide](https://pupvax.com/titer-testing/) — understand common titer testing questions before asking a vet.
- [PupVax](https://pupvax.com/) — puppy and dog vaccination schedule reference.

## Other Tools

- [iWorkViewer](https://iworkviewer.com/) — open Apple Pages, Numbers, and Keynote files on Windows.
- [InvoicePad](https://invoicepad.net/) — free invoice generator with no sign-up and no watermark.
- [PlantingCalendar](https://plantingcalendar.net/) — planting dates by crop and zone.
- [BabyPercent](https://babypercent.com/) — baby growth percentile calculator.

These tools are small static sites. Most processing happens in the browser. The goal is to solve one narrow problem per page with no account wall.
