# Free Browser Tools 2026

This note tracks a small set of focused, privacy-first browser tools. Each tool is designed around one narrow task and can be used without creating an account.

## File and Photo Utilities

- [iWorkViewer](https://iworkviewer.com/) helps open Apple Pages, Numbers, and Keynote files on non-Apple devices.
- [LivePhotoKit](https://livephotokit.com/) provides HEIC, LIVP, and Live Photo conversion tools that run in the browser.

## Planning and Productivity

- [ZonePlan](https://zoneplan.net/) helps compare time zones and schedule remote meetings.
- [InvoicePad](https://invoicepad.net/) is a lightweight invoice generator for freelancers.

## Everyday Calculators

- [FreeTDEE](https://freetdee.com/) includes TDEE, BMR, macro, and calorie deficit calculators.
- [BabyPercent](https://babypercent.com/) provides a baby growth percentile calculator.

## Home and Pet References

- [PlantingCalendar](https://plantingcalendar.net/) lists planting dates by crop and zone.
- [PupVax](https://pupvax.com/) tracks puppy and dog vaccination schedule references.
