# PupVax Broke Zero: A Better Puppy Deworming Schedule Page

PupVax reached its first Google Search click in the latest 28-day Search Console window: **1 click, 235 impressions, 0.4% CTR, average position 35.2**.

Today's single-variable experiment improves one page only:

- [Puppy Deworming Schedule: 2 Weeks to 6 Months](https://pupvax.com/puppy-deworming-schedule/)

The page now has:

1. A clearer age-by-age timeline from 2 weeks through adulthood.
2. FAQ content that answers the main search questions.
3. FAQ structured data.
4. Direct references to CAPC and AAHA veterinary guidance.
5. A stronger safety boundary: owners should not select products or calculate doses without a veterinarian.

The production page was rebuilt and deployed through GitHub Actions to Cloudflare Pages. The live route returns HTTP 200 and exposes the new title, H1, source links and FAQ schema.

## The lesson

A page being indexed is not the finish line. Search Console showed that PupVax was already receiving impressions. The better move was not to build another site, but to make one existing high-intent page more useful, more trustworthy and easier to understand.
